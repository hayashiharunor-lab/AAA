在学中の生徒管理を担当。卒業後は卒業履歴として保存し、必要なら別の人材管理システムへ橋渡しする(HR-MANAGER内では卒業後の人材管理は行わない)。

教育管理
学校管理という独立メニューは廃止し、教育管理の中で扱う。

🎓 教育管理 → 🏫 学校選択 → 👨‍🎓 クラス選択 → 👤 担当者選択 → 📚 教育業務
クラス中心設計:先生と生徒を直接紐付けしない(先生の異動・変更に対応するため)。

学校 → クラス → 担当者 → 生徒
担当者設計
「先生選択」ではなく「担当者選択」と呼ぶ。入力責任者(誰が入力したか・誰が確認責任を持つか)を記録するため。 教育分析では入力者ではなく「クラス・授業担当・教科」を基準にする。

外部連携管理
提携教育機関・受入企業を管理。将来的に企業担当者の名刺AI読み込み構想あり (名刺画像アップロード → AI文字認識 → 連絡先自動入力、または画像保存＋手入力)。
